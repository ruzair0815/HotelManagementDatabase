# Hotel Management Database System

This project is a hotel management system that allows customers and employees to create, manage, and view reservations. It simulates real-world hotel booking operations using a relational database and a web-based interface.

## Features

- Create and manage hotel reservations
- Store and retrieve customer and employee data
- View booking details and reservation history
- Manage check-in and check-out dates
- Calculate billing information for reservations

## Technologies

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Local Testing & Execution:** XAMPP / Apache Server, MySQL CLI
